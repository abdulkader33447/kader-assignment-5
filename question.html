Question-1: What are the different ways to select an element in the DOM?

The different ways to select an element in the DOM is query selector, tag selector, id selector, class selector

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML is a js property use for adding HTML element, Inner text is what is inside a text element , textContent is a JavaScript property used to read the text content of an HTML element.

Question-3: What is event delegation in the DOM?

Used to apply the same event to many elements.

Question-4: What is event bubbling in the DOM?

Bubbling is the process of propagation. When an event occurs on a child element, it spreads like a bubble throughout the document.

Question-5: How do you create, add, and remove elements using JavaScript?

Create an element with 'document.createElement()' . Add element by 'appendChild()' . Remove elements with 'remove()'' or 'removeChild()'
